## Operating Systems

## Algorithms

[Numerical Methods](http://www.teampli.net/RobinV/numeric.htm) - Implementation of various numerical and statistical methods.

[Physics Computations](https://www.plasmaphysics.org.uk/programs/) - Computer Programs for Plasma Physics, Atomic Physics and Radiative Transfer.

## Toolkit

* [Iron Spring PL/I](http://www.iron-spring.com) - The leading PL/I compiler for Linux.
* [PLI-2000](https://github.com/Steadsoft/PLI-2000) - A PL/I compiler for Windows NT.
* [PL/I Language Support](https://github.com/zowe/zowe-pli-language-support) - Enhances the PL/I programming experience on your IDE, available as a VS Code extension.
* [IBM Z Open Editor](https://ibm.github.io/zopeneditor-about/) - A free modern editing experience for IBM Z Enterprise languages.
* [SPF-Editor](https://github.com/michaelknigge/spf-editor) - Powerful editor and file manager that emulates the IBM mainframe ISPF editor.
* [SPFLite](https://www.spflite.com) - Another ISPF-like editor.
